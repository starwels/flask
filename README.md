python:3.8.1

#### Rodar:

```
pip install -r requirements.txt
```

#### Copiar o .env.template e adicionar as envs 

```
cp .env.template .env
```

#### Flask env
```
export FLASK_ENV=app.py
```

#### Database

1 - Criar o Database

2 - Rodar o comando
```
flask db upgrade
```

#### Rodar a aplicação
```
flask run
```
